🔧 How to Check Block Names in Braze

  1. Go to: Templates & Media → Content Blocks Library
  2. Look at the exact names of your 3 blocks
  3. Check for:
    • Underscores vs hyphens: css_styles vs css-styles
    • Capitalization: css_styles vs CSS_styles
    • Spaces: css styles (won't work!)
  4. Block names are case-sensitive and must match EXACTLY

Expected Block Names:

  • css_styles (with underscore)
  • hero_basic (with underscore)
  • footer (no underscore)

⚠️ Common Issues

Issue 1: Wrong Block Name

Problem: Block is named css-styles but code calls css_styles

Fix: Rename block in Braze OR change reference in code

Issue 2: Typo in Name

Problem: Block is hero_basic but you typed hero_baisc

Fix: Fix the typo

Issue 3: Block Empty

Problem: Block exists but has no content

Fix: Edit the block and add the HTML

Issue 4: Wrong Workspace

Problem: Blocks uploaded to different Braze workspace

Fix: Switch workspace or re-upload blocks

Issue 5: Block Not Saved

Problem: You created the block but didn't click "Save"

Fix: Open block and click "Save Content Block"
